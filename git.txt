git init - создаем git репозиторий из текущей папки
git add  . - просто пишем эту команду каждый раз перед commit
git commit -m message - создаем новую версию 
git push origin master - закидываем версию в github
git pull origin master - стягиваем новые изменения с Github
git clone ssh-ссылка - копируем репозиторий с Github
git log - посмотреть все коммиты


git branch - показывает все ветки (* обозначена ветка на которой мы находимся)
git branch название - создать новую ветку (просто создаем, но не переходим)
git checkout ветка - перейти на ветку
git checkout -b - создать и перейти 
git branch -D - удалить ветку
